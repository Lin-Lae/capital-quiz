## Features

---

## Run Locally
1. Clone: `git clone https://github.com/Lin-Lae/capital-quiz.git`  
2. Install: `npm install`  
3. Set `.env` with `PORT` & `DATABASE_URL`  
4. Run: `node index.js`  

Open `http://localhost:3000` in your browser.
